Ключевые понятия

Короткие пояснения, чтобы понять состояние системы без знания всех внутренних деталей.

Готовность
Показывает, готова ли система прямо сейчас принимать и обрабатывать новые chat-задачи.
Жизнеспособность
Показывает, отвечают ли базовые компоненты, даже если мощности для новых задач может не хватать.
Воркер
Процесс, который забирает задачу из dispatch и выполняет её.
Цель исполнения
Runtime endpoint, на который scheduler может допустить задачу.
Всего задач в очередях
Сколько задач ещё ждут admission/start и не начали выполняться.
Свободная мощность
Есть ли запас для новых chat-задач без ожидания в очереди.
Возраст heartbeat планировщика
Сколько времени прошло с последнего подтверждения, что scheduler жив и публикует heartbeat.
